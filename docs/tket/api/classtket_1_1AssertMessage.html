tket::AssertMessage::operator bool ( ) const

Always returns false, so that "|| AssertMessage() << a)" becomes "|| false)".

Also, stores the error message for later use by TKET_ASSERT macros; previously this information was passed on by exceptions, but that generated lots of code coverage branching problems.
