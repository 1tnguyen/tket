const std::vector< size_t > & tket::NeighboursInterface::operator() ( size_t  vertex)
virtual

Returns the neighbours of the given vertex.

The vector of neighbours is required to be stored internally. However, no guarantee that the reference will remain valid once another function call occurs. By default, throws (not implemented). (It's useful to be able to create a "null" object like this, because some algorithms don't actually need a neighbours object, but others do).

A sorted list of all adjacent vertices, stored internally.
