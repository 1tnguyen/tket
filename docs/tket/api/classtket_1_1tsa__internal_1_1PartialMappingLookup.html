Detailed Description

This is the same as ExactMappingLookup, except that we allow vertices not to have tokens.

It works simply by going through possible permutations of empty vertices and doing an exact permutation lookup (limiting the number of permutations to avoid excessive slowdown).

vertices_with_tokens_at_startEvery vertex mentioned within desired_mapping which has a token, just BEFORE the swaps are performed to enact the desired_mapping, must be mentioned here. Other vertices not mentioned in the mapping are allowed; they will simply be ignored.
