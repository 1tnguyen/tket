const Circuit & tket::CircPool::CCX_modulo_phase_shift ( )

Equivalent to CCX up to phase shift, using three CX.

Warning: this is not equivalent to CCX up to global phase so cannot be used as a direct substitution except when the phase reversal can be cancelled. Its unitary representation is like CCX but with a -1 at the (5,5) position.
