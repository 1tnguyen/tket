int tket::tsa_internal::get_move_decrease ( const VertexMapping &  vertex_mapping,
size_t  v1,
size_t  v2,
DistancesInterface &  distances 
)

For just the abstract move v1->v2, ignoring the token on v2, by how much does L (the total distances to home) decrease?

Parameters
vertex_mappingcurrent source->target mapping.
v1First vertex.
v2Second vertex. Not required to be adjacent to v1.
distancesAn object to calculate distances between vertices.
Returns
The amount by which L = get_total_home_distances would decrease, IF we moved the token on v1 to v2, IGNORING the token currently on v2 (which of course is impossible to do in reality if there is a token on v2), and leaving all other tokens unchanged. Doesn't have to be positive, of course, although positive numbers are good.
