typedef std::uint_fast16_t tket::tsa_internal::SwapConversion::EdgesBitset

Encodes a set of swaps, each one taken from the 15 possibilities.

With each swap given a numerical value from 1 to 15, we simply shift 1u by that amount (minus one), and OR them together. Thus, when looking up in a table, we only allow swap sequences whose edge bitset is a SUBSET of a given edges bitset (corresponding to the edges in the graph, i.e. allowed swaps).

typedef std::uint64_t tket::tsa_internal::SwapConversion::SwapHash

Encodes a sequence of <=16 swaps, each swap being one of the 15 possible swaps on vertices {0,1,2,3,4,5}, and hence encoded by 4 bits.

Zero represents the empty sequence.
